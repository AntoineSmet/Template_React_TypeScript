
# Template React
Template React TypeScript Vite Mui 

## Setup

1. Clone this repository

2. Navigate into the project directory

   ```bash
   $ cd 
   ```

3. Download modules 

   ```bash
   $ npm i
   ```
4. Run the app

   ```bash
   $ npm run dev
   ```
